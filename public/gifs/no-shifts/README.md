# No Shifts GIFs

This folder contains animated GIFs that randomly appear on the "No Shifts" page.

## How It Works

- Currently using a fallback GIF from Giphy
- Once you add GIFs to this folder, share them with me and I'll update the code to use them
- The app will randomly select one GIF each time the no shifts page loads
